# Proyecto Data Management and Business Intelligence 
Este proyecto tiene como objetivo gestionar y analizar grandes volúmenes de datos utilizando herramientas de Business Intelligence para tomar decisiones informadas.

## Características:
- Integración de bases de datos.
- Implementación de técnicas de análisis de datos, SSIS, SSAS, SSRS.
- Generación de reportes
## Tecnologías
- SQL Server
- Visual Studio 2025
- Git
